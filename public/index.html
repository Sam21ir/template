Implementación de una infraestructura virtual con VMware vSphere

El proyecto consistió en la implementación de una infraestructura virtual utilizando VMware vSphere, con tecnologías como VMware ESXi y vCenter Server. El objetivo fue optimizar el uso de los recursos, garantizar la alta disponibilidad de las aplicaciones críticas y facilitar la gestión centralizada de la infraestructura. Entre las características clave se encuentran la creación de máquinas virtuales, la implementación de alta disponibilidad (HA), la migración en tiempo real de máquinas (vMotion) y la optimización del rendimiento mediante el uso compartido de recursos.
